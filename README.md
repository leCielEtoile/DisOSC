# DisOSC

## Overview
VRChatのミュート状態をDiscordと同期させるツールです。
VRChatでイベント等をしている際、裏でDiscordを通して繋がっているときにVRChat上でDiscordのマイクミュートを制御できればいいなと思って作りました。

## Requirement
Node.js 20.11
Windows 10
